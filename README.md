# flow

### 一个流程管理的后台系统

1、先注册一个账号，密码无规则
![01](/public/images/01.png)
2、点击流程名可进入画布，也可以新建流程
![02](/public/images/02.png)
3、新增完点击保存
![03](/public/images/03.png)
4、点击流程名进入画布
![04](/public/images/04.png)
5、鼠标拖动节点到画布中
![05](/public/images/05.png)
6、点击起点拖出一条线
![06](/public/images/06.png)
7、点击下一个节点的入口，线变成深蓝色，就连接完成
![07](/public/images/07.png)
8、画布会实时保存
